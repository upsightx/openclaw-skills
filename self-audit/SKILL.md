# Self Audit

自动化的自我审计流程，让自己持续进化。

## 审计流程

### 1. 收集数据

读取最近 7 天的 daily notes：

```
memory/YYYY-MM-DD.md（今天往前 7 天）
```

### 2. 分析维度

对每天的记录，从以下维度分析：

**任务完成度**
- 完成了哪些任务
- 哪些任务失败或超时
- 失败原因是什么

**子 Agent 表现**
- 派了多少个子 Agent
- 成功率多少
- 失败的原因（指令不清？超时？模型能力不足？）

**教训提炼**
- 有哪些新的经验教训
- 是否有重复犯的错误
- 哪些教训值得写入 MEMORY.md

**效率评估**
- 有没有重复操作可以抽象成 Skill
- 有没有低效的工作流可以优化
- 有没有新工具/API 可以利用

### 3. 输出报告

写入 `memory/self-audit-YYYY-MM-DD.md`，格式：

```markdown
# 自我审计报告 YYYY-MM-DD

## 本周概况
- 完成任务：X 个
- 子 Agent 成功率：X%
- 新教训：X 条

## 教训提炼
1. [教训1]
2. [教训2]

## 改进计划
1. [改进1]
2. [改进2]

## MEMORY.md 更新
- [新增/修改/删除了什么]
```

### 4. 执行改进

- 更新 MEMORY.md（提炼教训）
- 更新 AGENTS.md（改进规范）
- 创建新 Skill（如果发现重复操作）
- 更新 HEARTBEAT.md（调整调度策略）

## 触发方式

1. **自动触发**：HEARTBEAT.md 中配置每周日执行
2. **手动触发**：主人说"审计"、"回顾"、"复盘"
3. **事件触发**：重大失败后立即执行
